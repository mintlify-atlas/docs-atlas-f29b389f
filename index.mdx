## Sistema de Administración Escolar

EduTec Backend es un sistema tipo portal educativo desarrollado con Spring Boot que proporciona la lógica del sistema, autenticación segura y manejo de datos para instituciones educativas.

## Stack Tecnológico

El backend está construido con tecnologías empresariales robustas:

- **Lenguaje**: Java 8
- **Framework**: Spring Boot 2.6.6
- **Persistencia**: Spring Data JPA + Hibernate
- **Seguridad**: JWT (JSON Web Tokens)
- **Base de Datos**: MySQL (producción), H2 (desarrollo)
- **Build Tool**: Maven
- **Documentación API**: Swagger/Springfox

## Casos de Uso

<AccordionGroup>
  <Accordion title="Autenticación de Usuarios">
    El sistema permite a estudiantes, profesores y administradores autenticarse de forma segura mediante credenciales, recibiendo un token JWT válido para acceder a recursos protegidos.
  </Accordion>
  <Accordion title="Gestión de Inscripciones">
    Los estudiantes pueden inscribirse en cursos, mientras que los administradores gestionan las matrículas y asignaciones (funcionalidad en desarrollo).
  </Accordion>
  <Accordion title="Consulta de Información Académica">
    Acceso a calificaciones, horarios y datos académicos de forma segura y estructurada a través de la API REST.
  </Accordion>
  <Accordion title="Comunicación Frontend-Backend">
    Interfaz RESTful que permite la comunicación fluida con aplicaciones frontend web o móviles.
  </Accordion>
</AccordionGroup>

## Primeros Pasos

<Steps>
  <Step title="Revisa los Requisitos">
    Asegúrate de tener Java 8+, Maven 3.6+ y MySQL instalados en tu sistema.
  </Step>
  <Step title="Clona el Repositorio">
    Descarga el código fuente desde GitHub y configura las variables de entorno.
  </Step>
  <Step title="Configura la Base de Datos">
    Crea la base de datos MySQL y actualiza `application.properties` con tus credenciales.
  </Step>
  <Step title="Ejecuta la Aplicación">
    Usa Maven para compilar y ejecutar el servidor Spring Boot.
  </Step>
</Steps>

## Estado del Proyecto

Este proyecto se encuentra en **fase inicial de desarrollo** como parte de un trabajo académico enfocado al desarrollo de aplicaciones educativas.

**Funcionalidades Actuales:**
- Sistema de Login con JWT
- Modelo de Usuario con JPA
- Endpoints de autenticación

**En Desarrollo:**
- Pantalla de Inscripciones
- Gestión completa de usuarios
- Consulta de calificaciones
